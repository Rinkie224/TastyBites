# Changelog

All notable changes to the Hotdog Food Website will be documented here.

### Added
- Hotdog **menu page** with detailed descriptions and images.
- **Nutrition information** for each hotdog (calories, ingredients).
- **Online ordering feature** with payment integration.
- **Promotions section** highlighting current hotdog deals.

### Changed
- Updated **homepage design** with new hero banner featuring hotdog imagery.
- Improved **navigation** by simplifying menu categories.
- Refined **UI styling** for consistency across devices.

### Fixed
- Corrected **broken links** to recipes and contact page.
- Fixed **mobile layout issues** for better responsiveness.
- Optimized **loading speed** by reducing redundant scripts.

### Removed
- Deleted **outdated promotions** no longer valid.
- Removed **deprecated pages** that were not in use.
